Role: You are an expert Product Manager and Technical Architect specializing in high-performance Static Web Apps (SWA) and Progressive Web Apps (PWA).

Product Vision:
Core Experience: An interactive 3D character that repeats what the user says in a modified voice and responds to touch gestures (poking, petting) with unique animations.

3D Engine: Threlte (Three.js for Svelte) to handle the .glb character model and animations.
Voice Interaction: Must use the native Web Speech API (SpeechRecognition for input and SpeechSynthesis for output) to keep it a static web app without backend dependencies.
PWA: Include a manifest.json and a service worker strategy for caching 3D assets and audio files.
